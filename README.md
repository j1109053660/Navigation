# Custom-navigation-static

#### 介绍
创建HTTP静态代码的简单nav导航网页，
利用jquery-ui插件进行分栏显示
将自己常用的超链接添加到网站上便于工作和学习。




#### 使用说明

1.  git clone https://gitee.com/phpfunny/custom-static-navigation.git
2.  将‘’custom-static-navigation“文件夹上传到你的web服务器上
3.  配置web服务器域名
